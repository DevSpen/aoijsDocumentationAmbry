# $onlyNSFW

This function only allows the command to be ran in NSFW marked channels

```text
$onlyNSFW[error when ran in non-nsfw channel]
```

```javascript
bot.command({
name: "special",
code: `Special Command
$onlyNSFW[Command only executable in NSFW channels!]`
})
```
